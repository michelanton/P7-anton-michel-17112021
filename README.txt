
Readme : 
Réalisation d'un réseaux social pour l'entreprise Groupomania.
Réalisé avec MySQL pour la base de données, Vuejs pour le frontend et nodeJs pour le backend. 
Il vous faudra donc avoir installer ces pré-requis (MySQL, VUEJS, NodeJS, GIT) pour faire fonctionner l'application.
- GIT :   https://github.com/michelanton/P7-anton-michel-17112021
https://github.com/michelanton/P7-anton-michel-17112021.git
	`git clone git@github.com:marieparet/groupomania.git`
- Backend : 
	dans le terminal : 
		npm install 
		puis  node server   (port: 3000)
Renseignez votre Utilisateur et votre mot de passe de base de donnée dans le fichier .env (dotenv).
- Frontend : 
	dans le terminal : 
		npm install
		puis,  npm run serve  (port 8080).
- Base de données :
	Importez le document groupomania_DataBase.sql dans MySQL.
